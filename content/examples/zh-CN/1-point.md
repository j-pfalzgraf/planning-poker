## 📋 示例 1：替换图标

### 用户故事

> 作为**用户**，我希望**看到现代化的保存图标**，以便**界面看起来更符合时代**。

### 背景

"保存"按钮目前使用软盘图标（`floppy-disk`）。由于软盘已不再常见，应将其替换为勾选图标。

### 技术细节

```text
受影响文件: app/components/SaveButton.vue
图标库: @heroicons/vue
旧图标: FloppyDiskIcon
新图标: CheckIcon
```

### 验收标准

- [ ] 在 `SaveButton.vue` 中将图标从 `FloppyDiskIcon` 替换为 `CheckIcon`
- [ ] 更新图标导入
- [ ] 尺寸保持 `w-5 h-5`
- [ ] 在桌面端和移动端进行视觉检查
- [ ] 无功能变更

### 完成定义

- [ ] 代码审查已完成
- [ ] 在预发布环境测试通过
- [ ] 截图已记录在工单中

## 📋 示例 2：添加工具提示

### 用户故事

> 作为**客户**，我希望**了解价格是否含税**，以便**在结账时不会感到意外**。

### 背景

价格字段旁边的信息图标（`ℹ`）应在悬停时显示带有"含增值税"文字的工具提示。项目中已存在工具提示组件。

### 技术细节

```vue
<template>
  <Tooltip text="含增值税">
    <InfoIcon class="w-4 h-4 text-gray-400 cursor-help" />
  </Tooltip>
</template>
```

### 验收标准

- [ ] 使用 `@/components/ui/Tooltip.vue` 中的工具提示组件
- [ ] 文字："含增值税"
- [ ] 工具提示在悬停和聚焦时出现（无障碍访问）
- [ ] 延迟：显示前 200ms
- [ ] 位置：顶部居中

## 📋 示例 3：根据样式指南调整 CSS

### 用户故事

> 作为**品牌经理**，我希望**将按钮颜色调整为新的样式指南**，以便**所有产品看起来一致**。

### 背景

主按钮使用 `#0066cc`，根据新样式指南应为 `#0052a3`。

### 技术细节

| 属性   | 旧值      | 新值      |
| ------ | --------- | --------- |
| 背景色 | `#0066cc` | `#0052a3` |
| 悬停   | `#0055b3` | `#003d7a` |

**文件：** `tailwind.config.ts`

```typescript
primary: {
  500: '#0052a3', // 原值: #0066cc
  600: '#003d7a', // 原值: #0055b3
}
```
