# 2 Story Points – Небольшие изменения

> **Трудозатраты:** Полдня
> **Риск:** Низкий до умеренного
> **Тесты:** Рекомендуются юнит-тесты

---

## Пример 1: Новое поле формы

**Название:** Опциональное поле номера телефона в контактной форме

**Описание:**
В контактную форму должно быть добавлено опциональное поле ввода номера телефона. Валидация формата для российских/международных номеров.

**Критерии приёмки:**

- [ ] Новое поле `phone` в форме
- [ ] Метка: "Телефон (необязательно)"
- [ ] Валидация: Разрешены только цифры, +, -, пробелы
- [ ] Поле отправляется на бэкенд

---

## Пример 2: Добавление сортировки

**Название:** Сортировка списка товаров по цене

**Описание:**
В обзор товаров должна быть добавлена опция сортировки "Цена по возрастанию/убыванию".

**Критерии приёмки:**

- [ ] Выпадающий список с опциями сортировки
- [ ] Сортировка на стороне клиента
- [ ] Активная сортировка визуально выделена

---

## Пример 3: Диалог подтверждения

**Название:** Показ диалога подтверждения при удалении

**Описание:**
Перед удалением записи должно появляться модальное окно: "Вы уверены, что хотите удалить?"

**Критерии приёмки:**

- [ ] Модальное окно с "Да" / "Отмена"
- [ ] Удаление только при подтверждении
