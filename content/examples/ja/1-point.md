## 📋 例 1: アイコンの置き換え

### ユーザーストーリー

> **ユーザー**として、**モダンな保存アイコンを見たい**。そうすることで、**UIがより現代的に見えます**。

### 背景

「保存」ボタンは現在フロッピーディスクアイコン（`floppy-disk`）を使用しています。フロッピーディスクは一般的ではなくなったため、チェックマークアイコンに置き換える必要があります。

### 技術詳細

```text
対象ファイル: app/components/SaveButton.vue
アイコンライブラリ: @heroicons/vue
旧アイコン: FloppyDiskIcon
新アイコン: CheckIcon
```

### 受け入れ基準

- [ ] `SaveButton.vue` のアイコンを `FloppyDiskIcon` から `CheckIcon` に置き換える
- [ ] アイコンのインポートを更新する
- [ ] サイズは `w-5 h-5` のまま
- [ ] デスクトップとモバイルで視覚的に確認する
- [ ] 機能的な変更なし

### 完了の定義

- [ ] コードレビュー完了
- [ ] ステージング環境でテスト済み
- [ ] チケットにスクリーンショットを記録

## 📋 例 2: ツールチップの追加

### ユーザーストーリー

> **お客様**として、**価格に消費税が含まれているかどうかを理解したい**。そうすることで、**チェックアウト時に驚かずに済みます**。

### 背景

価格欄の横にある情報アイコン（`ℹ`）にマウスを乗せると、「税込」というテキストのツールチップを表示する必要があります。ツールチップコンポーネントはプロジェクトに既に存在します。

### 技術詳細

```vue
<template>
  <Tooltip text="税込">
    <InfoIcon class="w-4 h-4 text-gray-400 cursor-help" />
  </Tooltip>
</template>
```

### 受け入れ基準

- [ ] `@/components/ui/Tooltip.vue` のツールチップコンポーネントを使用する
- [ ] テキスト: 「税込」
- [ ] ホバーとフォーカスでツールチップを表示する（アクセシビリティ）
- [ ] 遅延: 表示まで200ms
- [ ] 位置: 上部中央

## 📋 例 3: スタイルガイドに基づくCSS調整

### ユーザーストーリー

> **ブランドマネージャー**として、**新しいスタイルガイドに合わせてボタンの色を調整したい**。そうすることで、**すべての製品が統一された外観になります**。

### 背景

プライマリボタンは `#0066cc` を使用していますが、新しいスタイルガイドによると `#0052a3` にする必要があります。

### 技術詳細

| プロパティ | 旧        | 新        |
| ---------- | --------- | --------- |
| 背景色     | `#0066cc` | `#0052a3` |
| ホバー     | `#0055b3` | `#003d7a` |

**ファイル:** `tailwind.config.ts`

```typescript
primary: {
  500: '#0052a3', // 以前: #0066cc
  600: '#003d7a', // 以前: #0055b3
}
```

### 受け入れ基準

- [ ] `tailwind.config.ts` の色を調整する
- [ ] すべてのプライマリボタンに影響（トークン経由で自動的に）
- [ ] WCAGのAAコントラスト比を維持（最小4.5:1）
- [ ] 機能的な変更なし
