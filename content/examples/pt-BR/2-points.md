# 2 Story Points – Pequenas Mudanças

> **Esforço:** Meio dia
> **Risco:** Baixo a moderado
> **Testes:** Testes unitários recomendados

---

## Exemplo 1: Novo Campo de Formulário

**Título:** Campo Opcional de Telefone no Formulário de Contato

**Descrição:**
Um campo de entrada opcional para número de telefone deve ser adicionado ao formulário de contato. Validação de formato para números brasileiros/internacionais.

**Critérios de Aceite:**

- [ ] Novo campo `phone` no formulário
- [ ] Rótulo: "Telefone (opcional)"
- [ ] Validação: Apenas números, +, -, espaços permitidos
- [ ] Campo é enviado para o backend

---

## Exemplo 2: Adicionar Opção de Ordenação

**Título:** Ordenar Lista de Produtos por Preço

**Descrição:**
Uma opção de ordenação "Preço crescente/decrescente" deve ser adicionada à visão geral de produtos.

**Critérios de Aceite:**

- [ ] Dropdown com opções de ordenação
- [ ] Ordenação no lado do cliente
- [ ] Ordenação ativa destacada visualmente

---

## Exemplo 3: Diálogo de Confirmação

**Título:** Mostrar Diálogo de Confirmação ao Excluir

**Descrição:**
Antes de excluir uma entrada, um modal deve aparecer: "Tem certeza que deseja excluir?"

**Critérios de Aceite:**

- [ ] Modal com "Sim" / "Cancelar"
- [ ] Exclusão apenas na confirmação
